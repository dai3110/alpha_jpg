下記取説をお読みいただいた上でご利用ください

注意点

  • このツールは、この画面を表示する以外の通信を行いません。画像の生成から保存工程まで、すべてフロントサイドで完結しています。
    そのため、入力した画像が外部に漏れることはありませんが、念のため機密性の高いデータの使用は避けてください。
  • 生成されたZIPファイルを展開すると、xxx.jpg, xxx_mask.jpg, index.html, alpha.js の４ファイルが含まれます。
    canvas要素の特性上、ローカルでそのままブラウザで確認すると、(少なくとも)chromeではJavaScriptのエラーで動作しません。localhostなど、Webサーバーを経由して確認してください。
